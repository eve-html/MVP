# Требования
Установленные Node.js и npm

## Инструкция
1. Установите зависимости <br>
Перейдите в папку с сервером и выполните:
```
cd server
npm install
```
2. Запустите сервер <br>
В той же папке выполните:
```
npm run dev
```
3. Откройте сайт
Перейдите в браузере по адресу: <http://localhost:3000>
